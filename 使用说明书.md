# 大连海洋大学学生信息采集器：使用说明书

## 1. 这是什么

这是一个只用于查看大连海洋大学官网公开信息的 Python 小程序。它帮你把分散在官网中的新闻、通知和公开附件链接整理成本地文件，方便集中查看。

它不会登录网站，不会访问第三方网站，也不会绕过统一身份认证。如果某篇通知的正文需要登录，它会保留公开列表中的标题和原文链接，并提示正文没有采集。

## 2. 最简单的用法：双击启动

1. 在本项目文件夹中找到 [一键启动采集器.bat](一键启动采集器.bat)。
2. 双击它。
3. 稍等片刻。程序会以每次至少间隔 1 秒的速度访问网站。
4. 完成后会自动打开 `output` 文件夹。
5. 优先打开 `report.md` 阅读；想用 Excel 查看时打开 `articles.csv`。

默认只采集每个栏目第 1 页，最多 10 篇文章，适合第一次使用。

## 3. 第一次使用前：安装 Python

如果启动窗口显示 `Python was not found`，说明电脑还没有安装 Python，或系统没有找到它。

1. 到 Python 官方网站下载安装 Python 3.10 或更高版本。
2. 安装第一页务必勾选 **Add Python to PATH**。
3. 安装完成后关闭并重新双击 `一键启动采集器.bat`。

不要把项目文件夹放进受权限限制的系统文件夹。当前放在 `E:\codex` 是合适的。

## 4. 结果文件说明

每次运行后，结果保存在 `output` 文件夹：

| 文件 | 推荐用途 | 内容 |
| --- | --- | --- |
| `report.md` | 直接阅读 | 每篇文章的标题、日期、原文链接、附件链接和正文摘要 |
| `articles.csv` | Excel 查看、筛选 | 每篇文章一行，可按栏目或日期筛选 |
| `articles.json` | 后续编程使用 | 最完整的结构化数据 |
| `files` 文件夹 | 仅使用下载附件选项后出现 | 下载成功的公开附件 |

再次运行会覆盖同一个 `output` 文件夹中的三份结果文件。如果想保留某次结果，请先把整个 `output` 文件夹复制或改名，例如改成 `output-2026-07-19`。

## 5. 如何采集更多内容

双击启动采用的是安全的小范围设置。如果你想多采一些，需要打开 PowerShell：

1. 在项目文件夹空白处按住 Shift 再点右键，选择“在终端中打开”。
2. 输入下面命令并按回车：

```powershell
python -m dlou_crawler --pages 3 --max-articles 60
```

参数含义：

| 参数 | 含义 | 建议 |
| --- | --- | --- |
| `--pages 3` | 每个栏目最多翻 3 页 | 初学者建议不超过 3 |
| `--max-articles 60` | 最多下载 60 篇文章正文 | 可按需要调整 |
| `--delay 1` | 两次请求之间至少等待 1 秒 | 不要设置小于 1 |
| `--output 路径` | 指定保存结果的位置 | 例如 `--output D:\我的学校通知` |
| `--download-files` | 下载文章中的公开附件 | 文件可能较大，请谨慎使用 |

下载附件的例子：

```powershell
python -m dlou_crawler --pages 1 --max-articles 20 --download-files
```

## 6. 常见问题

### 双击后窗口一闪而过

本项目的启动文件会在最后停住。若仍然一闪而过，请右键 `一键启动采集器.bat`，选择“编辑”，确认文件内容没有被其他软件改动；也可以在 PowerShell 中运行第 5 节的命令查看错误。

### 为什么有的文章没有正文或附件？

常见原因是网页本身要求统一身份认证、附件不是公开文件，或者网站临时无法访问。程序不会尝试绕过这些限制；请通过“原文”链接在学校网站中正常查看。

### Excel 打开 CSV 出现乱码怎么办？

程序使用 Excel 兼容的 UTF-8 编码。请在 Excel 中使用“数据 → 自文本/CSV”导入，并选择 UTF-8；较新的 Excel 直接双击通常也能正常显示。

### 为什么运行很慢？

程序故意让请求间隔至少 1 秒，以免影响学校网站。采集 10 篇文章通常需要几十秒，这是正常的。

## 7. 使用边界

- 仅采集网站公开内容，供个人学习和阅读使用。
- 不要降低请求间隔，不要频繁重复运行大规模采集。
- 不要将采集内容用于商业转载或批量传播。
- 不要尝试修改程序以绕过登录、验证码或访问限制。
